About

Saya adalah mahasiswa D3 Teknik Informatika Fakultas Ilmu Terapan Telkom University. Sersan adalah gaya hidup saya, artinya adalah serius santai. Web Developer menjadi ketertarikan saya di dunia pemograman, karena saya memiliki impian untuk membuat suatu bisnis yang memiliki aplikasi multiplatform diawal pengembangannya agar proses bisnis dapat berjalan dengan cepat tanpa terhalangi waktu pengembangan software sistem informasinya.

SPA (Single Page App) dan PWA (Progressive Web App) adalah teknologi yang akan saya gunakan untuk pengembangan web. Untuk itu teknologi seperti ReactJS, React Native dan PWA menjadi fokus utama saya dalam pembuatan website, bahasa pemograman javascript menjadi fondasi saya dalam belajar.
